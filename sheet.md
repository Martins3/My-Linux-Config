- git log --format="%h --> %B"

- tig -- path/to/dir : 看一个目录的 log

- git push -f <remote> <branch>
- git tag -d tagname
- git push --delete origin tagname

- -> 如何将多个 commit squash 一下

- git reset --soft HEAD^
- -> 拉取 tags
- git fetch --tags
- -> 在一个特定的 commit 上打 tag

### git patch

- git format-patch -1 -v # 说明 patch 的版本
